Windows系统为电脑设置固定ip的方法

2019年03月18日 99 字 教程整理 服务器
目录
  1. 1. 查看和记录以太网状态的详细信息
  2. 2. 修改IPv4属性

目
录

很多时候，我们在一个固定地方办公，电脑连接了打印机和其他设备，或是共享文件夹，前后端接口调用；每次关机开机后ip地址就会重置，因此我们需要要将ip地址固定。

1. 查看和记录以太网状态的详细信息

单击【控制面板】-【网络和 Internet】，【打开网络和共享中心】；点击【本地连接】-【详细信息】，即可看到IP地址、子网掩码、默认网关、DNS服务器信息。
将上述IPv4地址、IPv4子网掩码、IPv4默认网关和IPv4-DNS服务器信息记录下来，之后的步骤会用到。

2. 修改IPv4属性

点击【属性】，双击【Internet协议版本 4(TCP/IPv4)】 。

依次填入上一步获得的IP地址、子网掩码、默认网关和首选DNS服务器/备用DNS服务器等信息。

输入完成后点击【确定】即可。

点击网络属性窗口的【确定】，然后等待网络重新连接。

此时PC的ip地址已成功设置为固定值。
