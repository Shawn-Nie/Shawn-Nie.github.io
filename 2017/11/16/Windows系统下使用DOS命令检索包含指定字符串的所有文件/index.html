Windows系统下使用DOS命令检索包含指定字符串的所有文件

教程整理
  • CMD
  • DOS
  • Findstr
  • Windows
  • 字段检索

在windows系统下，来查找并修改指定目录下包含某一字符串的所有文件，麻烦又费时。其实系统在DOS命令中，提供了Findstr命令来查找指定的一个或多个文件文件中包含（或通过参数 /V来控制不包含）某些特定字符串的行，并将该行完整的信息打印出来，或者打印查询字符串所在的文件名。

打开cmd，输入help findstr命令即可查看Findstr命令的语法详解。

Findstr命令的具体用法如下：

进入即将检索文件所在的文件夹

具体命令: cd <目录路径>
如：
进入C:\Users\HP\workspace文件夹，则输入：C:\Users\HP\workspace 即可进入。

使用findstr命令，检索包含指定字符串的文件或内容

命令: findstr /s /i "<字符串内容>" *.* ；
如：
findstr /s /i "Project_info" *.* ,即可输出C:\Users\HP\workspace目录下包含”Project_info”字符串的所有文件。
