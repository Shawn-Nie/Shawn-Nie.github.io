腾讯云服务器Ubuntu系统设置允许root用户登录

教程整理
  • Linux
  • Ubuntu
  • 腾讯云

腾讯云主机ubuntu系统默认用户名为ubuntu。在默认的ubuntu用户权限下，很多系统操作（如编辑/删除/上传文件等）都不被允许的。因此，为了更方便的使用系统功能，通常选择root账户直接登录。

启用root用户登录的步骤如下：

修改root密码

具体命令: sudo passwd root
在窗口中输入及确认新的root用户密码，即可生效。

修改ssh配置

命令: sudo vi /etc/ssh/sshd_config ；
找到 PermitRootLogin 这项 将其改为 yes 。

修改完成后，保存退出并重启ssh服务既可。
sudo service ssh restart
