Ubuntu下安装Maven

教程整理
  • Linux
  • Maven
  • Ubuntu

环境需求

安装Maven前，先确保系统已正常安装JDK。

安装步骤

下载并解压安装文件

  1. 输入wget http://apache.fayea.com/maven/maven-3/3.5.0/binaries/apache-maven-3.5.0-bin.tar.gz命令，下载maven程序安装包。
    如apache地址访问较慢，可尝试从镜像网址获取下载文件：wget http://mirrors.tuna.tsinghua.edu.cn/apache/maven/maven-3/3.3.9/binaries/apache-maven-3.3.9-bin.tar.gz。
  2. 解压Maven安装文件
    tar -xzf apache-maven-3.5.0-bin.tar.gz
  3. 移动Maven到/usr/local中
    mv apache-maven-3.5.0 /usr/local/apache-maven-3.5.0

设置Maven环境变量

  1. 编辑~/.bashrc文件
    输入vi ~/.bashrc命令，开始编辑文件
  2. 配置环境变量
    向文件添加如下语句：
    export M2_HOME=/usr/local/apache-maven-3.5.0
    export M2=$M2_HOME/bin
    export MAVEN_OPTS="-Xms256m -Xmx512m"
    export PATH=$M2:$PATH
  3. 保存~/.bashrc文件，使环境变量生效
    source ~/.bashrc

验证Maven安装

mvn --version
