1. ** 安装环境及文件准备 **

1.1. 系统环境

Ubuntu 16.04。

1.2. 查看系统是否已安装MySQL程序

登录系统终端，并输入 sudo netstat -tap | grep mysql 。
如系统没有返回信息，则没有安装，需按照下一章节进行安装；若如下显示，则说明系统已安装程序，无需再安装：

2. ** MySQL数据库主程序的获取与安装 **

2.1. 安装主程序

在终端输入：
sudo apt-get install mysql-server mysql-client
运行结果如下所示：

如在这一步出现: E: Unable to locate package mysql-server错误，先执行以下命令即可:
sudo apt-get update

2.2. root用户的密码设置

在此安装过程中会让要求输入root用户(管理MySQL数据库用户，非Linux系统用户)密码，按照要求输入即可。

2.3. 测试安装是否成功

在终端输入：
sudo netstat -tap | grep mysql
出现如下结果则安装成功：

也可通过登录MySQL测试：
在终端输入： mysql -uroot -p
接下来会提示输入密码，输入正确密码，即可进入程序。

3. ** MySQL的一些常用管理命令 **

3.1. 启动及停止MySQL服务

启动服务`sudo start mysql`
停止服务`sudo stop mysql`
重启服务`service mysqld restart`
一般情况下，MySQL会在服务器启动时也会自动启动服务。

3.2. 修改账户密码##

修改帐户密码 sudo mysqladmin -u <user> password <new password>

3.3. 设置远程访问

在初始情况下，mysql占用的3306端口只允许在IP 127.0.0.1上监听，拒绝了其他IP的访问（通过netstat可以查看到）。
如需允许其他ip远程访问数据库，则需取消本地监听，修改 my.cnf 文件：
命令：sudo vi /etc/mysql/my.cnf
找到如下命令行并注释掉：
bind-address = 127.0.0.1
